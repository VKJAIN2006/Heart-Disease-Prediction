❤️ Heart Disease Prediction Web App

A machine learning–based web application that predicts the likelihood of heart disease based on user health parameters.
The app is built using Python, Machine Learning, and Streamlit.

📌 Project Overview

Heart disease is one of the leading causes of death worldwide. Early prediction can help in timely diagnosis and prevention.
This project uses supervised machine learning algorithms to analyze medical data and predict whether a person is at risk of heart disease.

🚀 Features

📊 Real-time heart disease prediction
🧠 Trained using multiple ML algorithms
⚡ Fast and interactive UI using Streamlit
📈 High accuracy model (up to 87%)
🌐 Deployed and accessible online
🧠 Machine Learning Models Used

The following algorithms were trained and evaluated:

Logistic Regression ✅ (Selected)
K-Nearest Neighbors (KNN)
Naive Bayes
Decision Tree
Support Vector Machine (SVM)

📌 Final Model Chosen:
Logistic Regression, because it gave the highest accuracy (≈87%), better generalization, and stability.
